## ğŸ› Troubleshooting

**No sound?**
- Check files are in `resources/sounds/`
- Check files are named correctly
- Check they are MP3 or WAV format

**No avatars?**
- Check files are in `resources/avatars/`
- Check they are PNG, JPG, or JPEG
- Try square images (same width/height)

**Game won't start?**
- Make sure Java 11+ is installed
- Make sure JavaFX SDK is in the correct folder
- Run `run.bat` or `setup-and-run.ps1`
