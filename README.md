## Controls ğŸ®

-   **Space / Click**: Flap your wings!
-   **S**: Open the Settings menu
-   **A**: Cycle through different avatars
-   **1, 2, 3**: Switch up the sound effects
-   **R**: Restart instantly after you crash
-   **Esc**: Rage quit (I mean, exit the game)
